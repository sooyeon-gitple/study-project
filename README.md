## Gitple Study Project


````
#require node,yarn

$ yarn global add typescript

$ git clone https://github.com/sooyeon-gitple/study-project.git

$ cd server 
$ yarn start
$ cd ..
$ cd client
$ ng serve --open
````

- express: Server

- mongoDB(mongoose) : Database
